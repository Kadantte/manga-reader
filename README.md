# Manga Reader
Manga reader that scrapes manga sites for new chapters.  
Allows you to read chapters inside the application using Electron.

Currently supported sites:
- manganelo.com
- kkjscans.co
- webtoons.com
- hatigarmscanz.net
- 1stkissmanga.com
- mangakakalot.com
- mangadex.org
- mangakomi.com

Supported platforms:
- Windows
- Android
